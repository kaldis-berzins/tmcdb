I am making a postgresql database using prisma for EUIPO decisions. Can you generate a schema.prisma file for the following. Feel free to use more specific types if applicable. Everything marked as from euipo database should not be autogenerated in any way:

Proceeding (represents a whole proceeding through appeals):
- ID (generated unique id)
- CDDecision: string (cancellation division decision) (foreign key to decision table)
- BoADecision: string (board of appeals decision) (foreign key to decision table)
- GCDecision: string (same as above but for the general court)
- ECJDecision: string (same as above but for European Court of Justice)
- Trademark: number (foreign key to trademark table)

Trademark (represents a trademark):
- Number: number (primary key from euipo database)
- Type: string (trademark type from euipo database)
- Name: string (trademark name from euipo database)

Decision (represents a specific decision from one of the ruling bodies):
- ID: string (from euipo database)
- Institution: enum (consisting of CD, BoA, GC, ECJ)
- IngestDate: date (use the date when the row was created)
- RawText: string (raw text of decision document)
- Outcome: string (from euipo database)

Factor (represents a factor in a decision from a ruling body):
- ID: string (generated unique id)
- Decision: string (foreign key to decision table)
- Name: string (short factor description)
- Source: string (part of original decision that describes the factor)
